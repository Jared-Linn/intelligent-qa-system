# 智能问答系统 — 学习路线

> 从零搭建一个可运行的中文智能问答系统，边写代码边学技术

---

## 阶段划分

### 第1课：项目骨架与基础检索（本课）
- 搭建项目结构
- 找一个可用的中文问答数据集
- 实现 **TF-IDF + 余弦相似度** 的基础检索问答

### 第2课：问题理解模块
- 实现问题分类（基于规则 + 朴素贝叶斯）
- 实现关键词提取（TF-IDF / TextRank）
- 模块封装与接口设计

### 第3课：语义检索与向量化
- 使用 Sentence-BERT 将问题和文档编码为向量
- 使用 FAISS 实现语义检索
- 混合检索（BM25 + 向量）

### 第4课：检索增强生成（RAG）
- 基于检索结果 + 语言模型生成答案
- 接入本地或云端 LLM
- Prompt 设计与上下文组装

### 第5课：模型评价与优化
- EM/F1/BLEU/ROUGE 指标计算
- 检索和生成的效果分析
- 常见踩坑与优化策略

---

> ✅ **第1课目标**：跑通一个能回答问题的系统（哪怕很简陋）
